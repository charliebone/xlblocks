## Overview

XlBlocks is a powerful Excel add-in designed to extend the capabilities of Excel for data analysts, power users, and professionals in financial and business fields. Inspired by years of experience on financial trading desks, XlBlocks introduces advanced data structures and tools to make Excel a more robust platform for handling large datasets. These components can also be used to create interactive and dynamic spreadsheets.

---

## Key Features

- **Tables**: Perform advanced operations like filtering, sorting, joining, and aggregating on tabular data.
- **Lists**: Manage ordered collections of items with support for operations like adding, removing, and accessing elements.
- **Dictionaries**: Use key-value stores with support for various key types (numbers, strings, dates) and nested XlBlocks objects.
- **String Functions**: Leverage basic regex capabilities for pattern matching and text manipulation.
- **Interoperability**: Combine and manipulate data across tables, lists, and dictionaries seamlessly.

## Download and Installation

1. Visit the [XlBlocks GitHub Releases page](https://github.com/charliebone/xlblocks/releases).
2. Download the `.xll` file that matches your system's bitness:
   - **32-bit Excel**: Download `XlBlocks-AddIn-packed.xll`.
   - **64-bit Excel**: Download `XlBlocks-AddIn64-packed.xll`.
3. Follow the [installation guide](https://xlblocks.net/install/) for detailed instructions on adding the add-in to Excel.
